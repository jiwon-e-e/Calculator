스파르타 부트캠프
계산기 과제 


STEP 1.  클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기

- [v]  양의 정수(0 포함)를 입력받기
    - [v]  Scanner를 사용하여 양의 정수 2개(0 포함)를 전달 받을 수 있습니다.
    - [v]  양의 정수는 각각 하나씩 전달 받습니다.
    - [v]  양의 정수는 적합한 타입으로 선언한 변수에 저장합니다.

        
- [v]  사칙연산 기호를 입력받기
    - [v]  Scanner를 사용하여 사칙연산 기호를 전달 받을 수 있습니다.
    - [v]  사칙연산 기호를 적합한 타입으로 선언한 변수에 저장합니다.

- [v]  위에서 입력받은 양의 정수 2개와 사칙연산 기호를 사용하여 연산을 진행한 후 결과값을 출력하기
    - [v]  키워드 : `if` `switch`
    - [v]  사칙연산 기호에 맞는 연산자를 사용하여 연산을 진행합니다.
    - [v]  입력받은 연산 기호를 구분하기 위해 제어문을 사용합니다. (예를 들면 if, switch)
    - [v]  연산 오류가 발생할 경우 해당 오류에 대한 내용을 정제하여 출력합니다.
        - [v]  ex) “나눗셈 연산에서 분모(두번째 정수)에 0이 입력될 수 없습니다.“
                +) "숫자 입력은 정수만 가능하고, 기호입력은 주어진 범위 내에서 가능합니다"
            
- [v]  반복문을 사용하되, 반복의 종료를 알려주는 “exit” 문자열을 입력하기 전까지 무한으로 계산을 진행할 수 있도록 소스 코드를 수정하기
    - [ ]  키워드 : 무한으로 반복, 수정하기 (처음부터 무한 반복하는 것이 아니라, 위 스텝별로 진행하며 수정)
    - [v]  반복문을 사용합니다. (예를 들어, for, while…)
