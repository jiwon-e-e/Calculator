STEP 2. 클래스를 적용해 기본적인 연산을 수행할 수 있는 계산기 만들기

- [v]  사칙연산을 수행 후, 결과값 반환 메서드 구현 & 연산 결과를 저장하는 컬렉션 타입 필드를 가진 Calculator 클래스를 생성
    - [v]  사칙연산을 수행한 후, 결과값을 반환하는 메서드 구현
    - [v]  연산 결과를 저장하는 컬렉션 타입 필드를 가진 Calculator 클래스를 생성
    - [v]  1) 양의 정수 2개(0 포함)와 연산 기호를 매개변수로 받아 사칙연산(➕,➖,✖️,➗) 기능을 수행한 후 2) 결과 값을 반환하는 메서드와 연산 결과를 저장하는 컬렉션 타입 필드를 가진 Calculator 클래스를 생성합니다.

- [v]  STEP 1에서 구현한 App 클래스의 main 메서드에 Calculator 클래스가 활용될 수 있도록 수정
    - [v]  연산 수행 역할은 Calculator 클래스가 담당
        - [v]  연산 결과는 Calculator 클래스의 연산 결과를 저장하는 필드에 저장
    - [v]  소스 코드 수정 후에도 수정 전의 기능들이 반드시 똑같이 동작해야합니다.

- [v]  App 클래스의 main 메서드에서 Calculator 클래스의 연산 결과를 저장하고 있는 컬렉션 필드에 직접 접근하지 못하도록 수정 (캡슐화)
    - [v]  간접 접근을 통해 필드에 접근하여 **가져올** 수 있도록 구현합니다. (Getter 메서드)
    - [v]  간접 접근을 통해 필드에 접근하여 **수정할** 수 있도록 구현합니다. (Setter 메서드)
    - [v]  위 요구사항을 모두 구현 했다면 App 클래스의 main 메서드에서 위에서 구현한 메서드를 활용 해봅니다.

- [v]  Calculator 클래스에 저장된 연산 결과들 중 가장 먼저 저장된 데이터를 삭제하는 기능을 가진 메서드를 구현한 후 App 클래스의 main 메서드에 삭제 메서드가 활용될 수 있도록 수정
    - [v]  키워드 : `컬렉션`
        - [v]  컬렉션에서 ‘값을 넣고 제거하는 방법을 이해한다.’가 중요합니다!
